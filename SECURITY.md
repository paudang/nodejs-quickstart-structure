# Security Policy

## Supported Versions

We only support the latest version of `nodejs-quickstart-structure`. Please ensure you are using the most recent release.

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an email to phucdangb1400718@gmail.com. All security vulnerabilities will be promptly addressed.

Please include:
- A description of the vulnerability.
- Steps to reproduce.
- Potential impact.
