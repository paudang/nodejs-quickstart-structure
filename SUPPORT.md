# Support for nodejs-quickstart-structure

Need help with `nodejs-quickstart-structure`? We're here to help!

## Where to get help

### 1. Documentation
Please check the `README.md` and the `docs/` folder in the repository. We provide detailed information on usage, architecture choices, generated project structures, and various configuration combinations.

### 2. GitHub Issues
If you encounter a bug, have a feature request, or need help troubleshooting an issue related to the CLI, please check if an issue already exists.

- [Search existing issues](https://github.com/paudang/nodejs-quickstart-structure/issues)
- [Create a new issue](https://github.com/paudang/nodejs-quickstart-structure/issues/new)

When creating a new issue for a bug, please provide as much context as possible:
- Node.js version
- Operating System
- CLI choices made during generation (Architecture, Database, etc.)
- The exact error message or unexpected behavior

### 3. Public Roadmap
We maintain a public Trello board where you can track our current progress, see upcoming features, and vote for your favorites:
👉 **[View our Public Roadmap on Trello](https://trello.com/b/TPTo8ylF/nodejs-quickstart-structure-product)**

### 4. Tutorials & Articles
For deep-dive tutorials, architecture explanations, and updates, you can check out the related [Medium Article](https://medium.com/@paudang/nodejs-quickstart-generator-93c276d60e0b).

## Security Issues
If you discover a security vulnerability within `nodejs-quickstart-structure`, please do not open a public issue. Instead, reach out to the maintainer directly via the contact methods listed on their GitHub profile or npm profile. Security issues will be treated as a priority.
